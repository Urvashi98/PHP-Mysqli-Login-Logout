This project is deployed on xampp localhost. 
Place the project folder into directory/xampp/htdocs/project_directory.

The detailed tutorial will be updated soon.
Php and Mysqli procedural is used.

Code for Mysqli OO and MySqli Procedural is placed in the directories namely,LOGIN-LOGOUT-OO and LOGIN-LOGOUT-PDO. 
 
If you want to run only procedural method, then remove the oo and PDO folders to xampp/htdocs/oo or pdo.
